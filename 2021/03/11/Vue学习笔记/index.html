0x03 data & methods

在前面的学习中，已经初步意识到Vue支持许多强大的Property，本篇将介绍Vue中的两个Property：data和methods。

3.1 data

data是一个函数，通常在创建Vue app时被声明，其一般返回一个对象，这个对象中包含的字段可以直接被template访问：

通常情况下，开发者应该在data内声明此组件可能需要用到的变量名称，即使暂时用不到或后续会进行动态赋值（这种情况变量应该被赋予null或undefined）。将不包含在data中的变量添加进来是可行的，但是Vue不会跟踪它们，也就意味着不会动态响应。

3.2 methods

通过methods可以向Vue组件中添加方法，这些方法有可能在组件运行过程中被调用：

Vue 自动为 methods 绑定 this，以便于它始终指向组件实例。这将确保方法在用作事件监听或回调时保持正确的 this 指向。在定义 methods 时应避免使用箭头函数，因为这会阻止 Vue 绑定恰当的 this 指向。

这些 methods 和组件实例的其它所有 property 一样可以在组件的模板中被访问。在模板中，它们通常被当做事件监听使用：
